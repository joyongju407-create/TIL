# git 을 하기위한 오늘 배운 명령어

## 기본명령어

 1. cd : 경로이동
  -cd .. : 상위경로로 이동
 2. touch : 파일생성
 3. mkdir : 경로디렉토리생성
 4. mv : 파일 이동
 5. ls : 경로안에 있는 구성 확인
## git 명령어

 1. git add : 파일을 만들고 처음git서브에 등록대기
 2. git commit : 파일의 주석을 달고 최종 등록
 3. git clone : 깃허브 새 레보 커넥팅
 4. git status : 여러 정보확인
 5. git log,lg : 내가 깃허브서 작업한 로그보기
 6. git push origin main :  토큰 연결하기 push의 다른 용도는 잘모름
